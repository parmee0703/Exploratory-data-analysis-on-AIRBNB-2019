# Airbnb NYC 2019 Data Analysis & Clustering Project

This project performs **Exploratory Data Analysis (EDA)** and **Customer Segmentation using K-Means Clustering** on the Airbnb NYC 2019 dataset.

The analysis helps identify patterns in listing prices, reviews, and availability, while grouping listings into clusters based on their characteristics.

---

## 📌 Features

- Data Cleaning
- Missing Value Handling
- Duplicate Removal
- Outlier Detection
- Data Visualization
- Correlation Analysis
- K-Means Clustering
- Cluster-wise Insights

---

## 📂 Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

---

## 📊 Dataset

Dataset Used:

`AB_NYC_2019.csv`

The dataset contains Airbnb listing information such as:

- Price
- Number of Reviews
- Availability
- Room Type
- Location
- Host Information
